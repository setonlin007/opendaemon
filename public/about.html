核心理念

OpenDaemon 的宪法原则 — 为你而非替代你

🛡 用户主权

你的数据、你的规则、你的 AI。所有决策权始终在你手中，AI 是工具而非主人。

📦 本地优先

SQLite + 文件系统，无需云服务。数据从不离开你的机器，隐私天然有保障。

🔄 引擎无关

Claude、GPT、DeepSeek、Kimi 随意切换。不锁定任何供应商，自由选择最适合你的引擎。

📈 渐进增强

从简单聊天到自进化，按需开启。零配置即可上手，高级功能逐步解锁。

快速开始

五步启动你的自进化 AI

1

克隆仓库

git clone https://github.com/your-org/opendaemon.git
cd opendaemon
2

安装依赖

npm install
3

配置 API Key

# 复制示例配置
cp config.example.json config.json

# 编辑 config.json，填入你的 API Key
# 支持 Claude / OpenAI / DeepSeek / Kimi 等
4

启动服务

node server.mjs
5

打开浏览器

# 默认地址
http://localhost:3000
